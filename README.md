# Tic Tac Toe Game 
A simple two-player Tic Tac Toe game built using HTML, CSS, and JavaScript. Players take turns marking X and O on the grid until one wins or the game ends in a draw.

## Features
- Two-player gameplay
- Win and draw detection
- Restart functionality
- Turn-based system
- Clean and simple UI

## Run Locally
Open index.html in your browser
